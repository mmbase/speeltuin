dit zijn de gereviseerde versies van de form tags. eigenschappen van de nieuwe versies:
- geen dubbele code meer in de tag files.
- geen dubbel tags meer in form: en related: taglibraries.
- consistentere manier van interactie en parametrisering.
- documentatie!

overzicht.
Het is nu mogelijk om overal dezelfde form:<veldtype> tags te gebruiken. ook is er nog maar een createrelation tag, die zowel in de form:wizard als in de related:add tag
De createrelation tags: createrelation, checkboxrelation radiobuttonrelation kunnen velden in de context van hun body zetten, zodat field tags 
kunnen ontdekken wat de juiste actie of modifier moet zijn. zie (velden > waardes resolven)
