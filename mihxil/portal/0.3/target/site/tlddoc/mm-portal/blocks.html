mm-portal
Tag blocks

Produces a div, and takes the current node (_node), and determing the related 'blocks' (using 'blockposrel). For each found node then a div is produced, with the proper css classes to position them. In the body the variables 'block' and 'blockposrel' are produced, which are the two MMBase Nodes which are relevant. These blockposrel variable can typically be put into the portal:block tag.
